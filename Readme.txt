Barème et criètes d'évaluation:

- Interface utilisateur claire et bien pensée (4 points).
- Structure sémantique html et utilisation des méta pour optimiser le référencement (4 points).
- Intégration mobile first (4 points).
- Utilisation de git / branch / commit réguliers avec des message clairs et concis (5 points)
- Déploiement de la réalisation en ligne (3 points)


Pour giter mon projet en fonction de la demande de l'évaluation, je procède comme suit:

La branch Commit init - Dépôt de reset.css, index.html, style.css et des assets.
La branch Perm - Dépôt de permanences.html et permanence.css.
La branch Event - Dépôt de evenements.html et evenements.css.
La branch Contact - Dépôt de contact.html et contact.css.
La branch Main - Fusion des branches en une seule.
